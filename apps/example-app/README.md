# example-app

jp-ui を「利用者」として触るためのサンプルアプリです。
目的は UI の使い心地検証と、改善点のフィードバック収集です。

## 進め方

1. 画面/フローを 1 つずつ作る
2. 使いづらさや違和感があれば `feedback.md` に記録する
3. 記録を元にコンポーネント改善を進める

## ルール

- Base UI は原則ラッパー経由で利用する
- 例外や独自ロジックが必要なら先に相談する
- 使い心地の問題は小さくても記録する（積み上げ前提）

## 参照

- `feedback.md`: ログ
- `feedback-template.md`: テンプレート
- 実装: `src/pages/ExampleAppPage.tsx`
