# Templates

These files are starter snippets for jp-ui components. Copy into `src/components/...` and rename.

## Checklist for new components

- Add the component file(s) under `src/components/<component>/`.
- Add a Storybook file under the same folder.
- Update the barrel exports in `src/components/index.ts`.

## Files

- `base-ui-wrapper.tsx`: Base UI wrapper template.
- `custom-use-render.tsx`: Custom component template using `useRender`.
- `component.stories.tsx`: Storybook starter template.
